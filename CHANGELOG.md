# Changelog

All notable changes to this project will be documented in this file.

## [v0.3] - 2025-10-14

### Added

- Smart config file detection: automatically open config files when a directory contains only one config file and no directories
- List navigation cycling: up/down arrow keys now wrap around the list for seamless navigation
- Comprehensive test coverage for new features

### Changed

- Upgraded bubbletea from v0.25.0 to v1.3.10
- Upgraded bubbles from v0.18.0 to v0.21.0

### Improvements

- Consistent behavior for smart config detection in both root and subdirectories
- Better user experience with circular navigation through lists
- Enhanced test suite with edge case coverage

- Bugfix: select wrong item in list

## [v0.1] - Initial release
